إعداد Supabase — قاعدة البيانات

Supabase هي قاعدة بيانات PostgreSQL سحابية مجانية — ستحتفظ ببيانات الحملات والتبرعات

1

إنشاء حساب على Supabase

اذهب إلى supabase.com واضغط "Start your project" وسجّل بحساب GitHub الخاص بك.

📍 supabase.com 
┌─────────────────────────────────────────────────────────┐
│  🟢 Supabase                              [Start free]  │
├─────────────────────────────────────────────────────────┤
│                                                         │
│         Build in a weekend, scale to millions           │
│                                                         │
│    ┌─────────────────────────────────────────────┐     │
│    │  [Continue with GitHub]  ← اضغط هنا        │     │
│    └─────────────────────────────────────────────┘     │
│                                                         │
└─────────────────────────────────────────────────────────┘
2

إنشاء مشروع جديد

بعد تسجيل الدخول، اضغط "New Project" وأدخل البيانات:

📍 app.supabase.com → New Project 
┌─────────────────────────────────────────────────────────┐
│  New Project                                            │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  Organization:   [Personal]                             │
│                                                         │
│  Project name:   rafaqaa-v2          ← هذا            │
│                                                         │
│  Database Pass:  [اختار كلمة مرور قوية]  ← احفظها!   │
│                                                         │
│  Region:         ◉ Middle East (Bahrain)  ← الأقرب    │
│                                                         │
│              [Create new project]                       │
│                                                         │
└─────────────────────────────────────────────────────────┘
⚠️ مهم: احفظ كلمة مرور قاعدة البيانات — لن تُعرض مرة ثانية!
3

نسخ مفاتيح Supabase

من لوحة تحكم Supabase اذهب إلى Settings → API:

📍 Project Settings → API 
┌─────────────────────────────────────────────────────────┐
│  Project URL                                            │
│  ┌──────────────────────────────────────────────────┐  │
│  │  https://xxxxxxxxxxxx.supabase.co    [Copy] ←──  │  │
│  └──────────────────────────────────────────────────┘  │
│                                                         │
│  Project API Keys                                       │
│                                                         │
│  anon (public):                                         │
│  ┌──────────────────────────────────────────────────┐  │
│  │  eyJhbGciOiJIUzI1NiIsInR5cCI6Ikp...  [Copy] ←── │  │
│  └──────────────────────────────────────────────────┘  │
│                                                         │
│  service_role (secret):                                 │
│  ┌──────────────────────────────────────────────────┐  │
│  │  eyJhbGciOiJIUzI1NiIsInR5cCI6Ikp...  [Copy] ←── │  │
│  └──────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────┘

انسخ وأحفظ القيم الثلاث في ملف نصي:

# احفظ هذه القيم — ستحتاجها في الخطوات التالية SUPABASE_URL=https://xxxxxxxxxxxx.supabase.co SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1Ni... (المفتاح العام) SUPABASE_SERVICE_KEY=eyJhbGciOiJIUzI1Ni... (مفتاح الخدمة السري)
✅ انتهى Supabase! احتفظ بالمفاتيح الثلاثة للخطوة التالية.

4

إضافة متغيرات البيئة (Environment Variables)

افتح قسم "Environment Variables" وأضف القيم التالية واحدة واحدة:

📍 Environment Variables 
┌─────────────────────────────────────────────────────────┐
│  Environment Variables                                  │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  NAME                    VALUE                         │
│  ─────────────────────────────────────────────────     │
│  VITE_SUPABASE_URL      https://xxx.supabase.co  [Add] │
│  VITE_SUPABASE_ANON_KEY eyJhbGci...              [Add] │
│  VITE_API_URL           https://xxx.vercel.app   [Add] │
│                           ↑ ده رابط الـ API server     │
│                                                         │
└─────────────────────────────────────────────────────────┘
الاسم (NAME)القيمة (VALUE)
VITE_SUPABASE_URLرابط Supabase من الخطوة السابقة
VITE_SUPABASE_ANON_KEYالمفتاح العام (anon) من Supabase
VITE_API_URLرابط الـ API (اتركه فارغاً الآن)

ربط الدومين من Namecheap بـ Vercel

توجيه دومينك من Namecheap ليشير إلى موقعك على Vercel

1

إضافة الدومين في Vercel

من لوحة تحكم Vercel اذهب إلى مشروعك → Settings → Domains

📍 Vercel → Project → Settings → Domains 
┌─────────────────────────────────────────────────────────┐
│  Domains                                                │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  Add Domain:                                            │
│  ┌──────────────────────────────────┐                  │
│  │  www.rafaqaa.com    ← اكتب دومينك│  [Add]          │
│  └──────────────────────────────────┘                  │
│                                                         │
│  أو اكتب:  rafaqaa.com  (بدون www)                     │
│                                                         │
└─────────────────────────────────────────────────────────┘
2

نسخ DNS Records من Vercel

بعد إضافة الدومين، ستظهر لك سجلات DNS — انسخها:

📍 Vercel → Domains → DNS Configuration 
┌─────────────────────────────────────────────────────────┐
│  Configure DNS for rafaqaa.com                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  Add these DNS records to your domain provider:        │
│                                                         │
│  TYPE    NAME       VALUE                              │
│  ─────────────────────────────────────────────────     │
│  A       @          76.76.21.21       ← انسخ           │
│  CNAME   www        cname.vercel-dns.com  ← انسخ       │
│                                                         │
│  Status: ⏳ Pending DNS propagation...                 │
│                                                         │
└─────────────────────────────────────────────────────────┘
💡 ملاحظة: القيم الفعلية ستظهر في Vercel — انسخها من هناك لا من هنا!
3

│                                                         │
│  ثم أضف السجلات الجديدة:                               │
│                                                         │
│  [Add New Record]                                       │
│  ┌──────────────────────────────────────────────────┐  │

│  │ Value: 76.76.21.21  ← من Vercel                 │  │
│  │ TTL: Automatic                                   │  │

│  └──────────────────────────────────────────────────┘  │
│                                                         │
│  [Add New Record]                                       │
│  ┌──────────────────────────────────────────────────┐  │
│  │ Type: CNAME Record                               │  │
│  │ Host: www                                        │  │
│  │ Value: cname.vercel-dns.com  ← من Vercel        │  │

💡 التيب: استخدم القيم الظاهرة في Vercel وليس القيم في هذا الدليل — تختلف من مشروع لآخر!

التحقق من أن كل شيء يعمل

قائمة تحقق سريعة بعد انتهاء الإعداد

✓

قائمة التحقق النهائية

□  الموقع يفتح على الرابط الرئيسي (rafaqaa.com)
□  الصفحة الرئيسية تظهر بشكل صحيح مع البانرات
□  زر "تبرع الآن" يفتح نافذة التبرع
□  طرق الدفع الأربعة تظهر في نافذة التبرع
□  الدخول على /admin يعمل بمستخدم admin/admin123
□  لوحة التحكم تعرض التبرعات والحملات
□  الدومين يفتح بـ https (🔒 في المتصفح)
□  www.rafaqaa.com يحوّل لـ rafaqaa.com
!

مشاكل شائعة وحلولها

المشكلةالحل
الصفحة تظهر بيضاء تحقق من إعداد Root Directory في Vercel: artifacts/rafaqaa-website
خطأ "Failed to fetch" تحقق من قيمة VITE_API_URL في Vercel Environment Variables
الدومين لا يعمل انتظر 30 دقيقة أخرى — DNS يحتاج وقتاً للانتشار
خطأ "Invalid API Key" في Supabase تأكد من نسخ anon key كاملاً بدون مسافات
