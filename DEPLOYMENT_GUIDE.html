إعداد GitHub Repository

رفع الكود ورفع التحديثات

1

إنشاء ريبو جديد على GitHub

اذهب إلى github.com/new وأنشئ ريبو باسم rafaqaa-website

  • اختر Private للحفاظ على سرية الكود
  • لا تضيف README أو .gitignore (الكود جاهز)
2

إنشاء Personal Access Token

اذهب إلى GitHub Settings → Developer Settings → Personal Access Tokens → Generate New Token

  • اختر صلاحية repo (كاملة)
  • احفظ التوكن في مكان آمن
3

ربط Replit بـ GitHub ورفع الكود

في Replit، من نافذة الـ Version Control اختر "Connect to GitHub" أو استخدم الأوامر:

git remote add origin https://YOUR_TOKEN@github.com/USERNAME/rafaqaa-website.git

git push -u origin main

4

إعداد GitHub Secrets (للـ CI/CD)

  • Settings → Secrets and Variables → Actions
  • أضف جميع متغيرات البيئة كـ Secrets
▲

نشر الموقع على Vercel

ربط GitHub بـ Vercel للنشر التلقائي

💡 Vercel مناسب للـ Frontend فقط. الـ API Server (Express.js) يحتاج خادم منفصل مثل Railway أو Render.

1

إنشاء حساب Vercel

اذهب إلى vercel.com وسجّل بحسابك على GitHub مباشرة

2

Import المشروع

  • New Project → Import Git Repository
  • اختر ريبو rafaqaa-website
  • Root Directory: artifacts/rafaqaa-website
  • Framework Preset: Vite
  • Build Command: pnpm build
  • Output Directory: dist
3

إضافة Environment Variables في Vercel

  • Project Settings → Environment Variables
  • أضف: VITE_SUPABASE_URL و VITE_SUPABASE_ANON_KEY
  • أضف: VITE_API_BASE_URL = رابط الـ API Server
4

ربط الدومين الرسمي

  • Project Settings → Domains → Add Domain
  • أضف rafaqaa.org أو دومينك الرسمي
  • Vercel سيعطيك CNAME أو A Record تضيفه في DNS
🚂

نشر الـ API Server على Railway

الخادم الخلفي (Express.js + PostgreSQL)

1

إنشاء حساب Railway

اذهب إلى railway.app → New Project → Deploy from GitHub

2

إعداد المشروع

  • Root Directory: artifacts/api-server
  • Build Command: pnpm run build
  • Start Command: node dist/index.mjs
3

إضافة قاعدة البيانات

  • في Railway: New → Database → PostgreSQL
  • انسخ DATABASE_URL وأضفها كـ Environment Variable
  • شغّل: pnpm --filter @workspace/db push لإنشاء الجداول
4

أضف جميع Environment Variables

أضف كل المتغيرات المذكورة في قسم "متغيرات البيئة" أعلاه

إعداد Paymob (الدفع الإلكتروني)

بوابة الدفع الأكثر انتشاراً في مصر

✅ النظام مبرمج بالكامل لـ Paymob. كل ما تحتاجه هو الحصول على API Keys من حسابك.

1

إنشاء حساب Paymob

اذهب إلى accept.paymob.com → Sign Up كـ Merchant

  • ستحتاج: السجل التجاري للمؤسسة / البطاقة الضريبية
  • المدة: 1-3 أيام عمل للتفعيل
2

إنشاء Integration

  • Dashboard → Integrations → Add Integration
  • أنشئ Integration لـ Online Card Payment → احفظ الـ Integration ID
  • أنشئ Integration أخرى لـ Mobile Wallet (اختياري)
  • من كل Integration → Create Iframe → احفظ الـ Iframe ID
3

الحصول على API Key

  • Settings → Account Info → API Key
  • انسخه وأضفه كـ PAYMOB_API_KEY في Environment Variables
4

إعداد Callback URL في Paymob

  • Settings → Webhooks/Notifications
  • Transaction Processed Callback: https://your-api.com/api/payments/paymob/callback
🔌

أهم نقاط الـ API

Base URL: https://your-api.railway.app/api
