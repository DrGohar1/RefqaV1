إنشاء Supabase — قاعدة بيانات التبرعات

  • اذهب لـ supabase.com وسجّل دخول بحساب Google أو GitHub

إنشاء PostgreSQL على Neon.tech — قاعدة بيانات الأدمن

هنا بيتخزن حسابات الأدمن والجلسات — مجاني

  • اذهب لـ neon.tech وسجّل دخول
  • اضغط "New Project" — اختر منطقة قريبة من مصر مثل eu-central-1
  • بعد الإنشاء، اضغط على "Connection Details"
  • انسخ Connection String — يبدأ بـ postgresql://
console.neon.tech/app/projects/[project]/connection
Connection String (DATABASE_URL)
postgresql://user:password@ep-xxx.eu-central-1.aws.neon.tech/dbname?sslmode=require
💡 بديل: ممكن تستخدم Vercel Postgres — بعد نشر الموقع على Vercel اذهب لـ Storage → Create Database. هيديك نفس الـ DATABASE_URL تلقائياً.

رفع الكود على GitHub

شرط أساسي لنشر الموقع على Vercel

  • اذهب لـ github.com/new وأنشئ repository جديد — اسمه مثلاً rafaqaa-website
  • افتح Terminal/Command Prompt في مجلد المشروع
  • شغّل الأوامر التالية:
# ربط المجلد بـ GitHub git init git remote add origin https://github.com/اسمك/rafaqaa-website.git # رفع الكود git add -A git commit -m "Initial deploy" git push -u origin main
✅ بعد الرفع: هتشوف كل الملفات على GitHub — جاهزين للنشر.

النشر على Vercel

الموقع والسيرفر كلهم هينشروا على Vercel مجاناً

  • اذهب لـ vercel.com وسجّل دخول بحساب GitHub
  • اضغط "Add New → Project"
  • اختر الـ repository اللي رفعته (rafaqaa-website)
  • Vercel سيكتشف الإعدادات تلقائياً من ملف vercel.json
  • اضغط "Deploy" وانتظر 2-3 دقائق
vercel.com/new

إعداد Paymob — بوابة الدفع الإلكتروني

اختياري — للدفع بالبطاقات وفودافون كاش وإنستاباي

  • اذهب لـ accept.paymob.com وأنشئ حساب تاجر
  • اكتمل بيانات المؤسسة وانتظر الموافقة (1-3 أيام)
  • بعد الموافقة، اذهب لـ Settings → Account Info → انسخ API Key
  • اذهب لـ Payment Integrations → أنشئ Integration للبطاقة وانسخ Integration ID + Iframe ID
  • اذهب لـ Settings → Webhooks وأضف:
# Webhook URL في Paymob https://موقعك.vercel.app/api/payments/paymob/callback
💡 لو Paymob مش جاهز: الموقع يشتغل في وضع "Demo" — ممكن تأكد التبرعات يدوياً من لوحة التحكم.

إعداد Telegram Bot — إشعارات فورية

مجاني — موصى به جداً — تاخد إشعار لكل تبرع على الفور

  • افتح Telegram وابحث عن @BotFather
  • أرسل /newbot ثم اختر اسماً مثل "رفقاء البررة Bot"
  • احفظ الـ Token الذي سيظهر (مثل: 123456:ABC-DEF...)
  • أنشئ مجموعة أو قناة وأضف البوت إليها كـ Admin
  • لمعرفة Chat ID، أرسل رسالة في المجموعة ثم افتح:
https://api.telegram.org/botTOKEN/getUpdates # في الرد، ابحث عن: "chat": { "id": -1001234567890 }
بعدها: أضف TOKEN و Chat ID في معالج التثبيت (/install) أو في الإعدادات من لوحة التحكم.

اختبار الموقع

تأكد من عمل كل حاجة قبل الإطلاق

  • 🌐افتح الموقع — هيظهر splash screen ثم الصفحة الرئيسية
  • 📋تحقق من ظهور الحملات في الصفحة الرئيسية
  • 💰اضغط "تبرع الآن" على أي حملة وتأكد من ظهور Modal الدفع
  • 🔍افتح /track وجرّب بحث برقم تليفون أو Refqa ID
  • 🔐افتح /auth وسجّل دخول بـ admin/admin123
  • ⚙️في لوحة التحكم، تحقق من ظهور قسم "التبرعات والمدفوعات"
  • 📬جرّب تبرعاً تجريبياً وأكّده من لوحة التحكم — تحقق من وصول إشعار Telegram
  • 💳لو Paymob مُعدّ — جرّب دفعاً ببطاقة اختبار
